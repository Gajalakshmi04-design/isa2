Smart Q Door

The Smart Q Door revolutionizes the way you manage deliveries, providing a seamless, hands-free experience. By enabling remote access via the dedicated app, users can lock, unlock, and monitor their door from any location, ensuring secure package drop-offs even when they're not home. Built with user-centered design, this project delves into advanced mental models and UI principles, offering an intuitive and smooth interface that simplifies control. With enhanced safety features and real-time notifications, the Smart Q Door is more than just a door—it's a smart security solution tailored for modern living.
